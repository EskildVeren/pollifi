# pollifi

## Application prerequisites

### Node.js

This application requires Node.js in order to run. Install Node version 22.X using the guide on the official [Node.js webpage](https://nodejs.org/en)

### Firebase tools

Running this application also requires you to install the Firebase CLI, giving you access to the firebase command. This will allow you to run the emulators. The official firebase documentation has a [guide](https://firebase.google.com/docs/cli#mac-linux-auto-script) on how to install this, but below is a short guide on what is needed.

**Install Firebase CLI**  
Install the firebase CLI globally using `npm install -g firebase-tools`

**Log in to you firebase account**  
Run the `firebase login` command and follow the instructions that follows

## How to run the application

Clone the Pollifi repo  
`git clone https://github.com/EskildVeren/pollifi.git`

Enter the app folder  
`cd app`

Run the app  
`npm run dev`
